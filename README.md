# ReForge

> **Software is never finished—only abandoned.**

ReForge is an AI-powered software archaeology platform that discovers software worth preserving, understands why it mattered, restores what has been lost, and helps it continue evolving.

Every year, thousands of repositories become inactive. Some no longer build. Others depend on obsolete technologies. Many were abandoned not because the ideas failed, but because time moved on.

ReForge believes that software is more than source code. Every project represents ideas, experimentation, craftsmanship, and history. Before software is rewritten, migrated, or forgotten, it deserves to be understood.

Unlike traditional modernization tools that simply upgrade technology stacks, ReForge asks a different set of questions:

* What problem was this software solving?
* What made it valuable?
* Why was development abandoned?
* Does it deserve another chapter?

Using a team of autonomous AI agents, ReForge analyzes repositories, reconstructs their architecture and intent, restores functionality where possible, and proposes meaningful evolution while preserving the project's identity.

The goal is not to modernize every repository.

The goal is to ensure that valuable software is never lost simply because it was abandoned.
